NodeJS - the xmllint Package

Install

To get xmllint on your system, run the following to install:

npm install xmllint

Instantiation

Once you have it installed, you can start declaring it in your various js files like so:

var xmllint = require('xmllint');

Usage

This allows us to access functions elsewhere in our program using dot notation, like this:

var result = xmllint.validateXML({
xml: xmlData,
schema: schemaData,
},
function (err) { if (err) throw (err) }
);

After executing the above code, the result variable will indicate the status of the validation. If results.errors exists, the validation failed and the result.errors value will indicate why. If results.errors does not exist, then validation succeeded.

NOTE: the xmlData and schemaData variables must be strings of the file contents, not a file path.
