This demo will showcase some file sorting. The server has a couple directories of interest, chiefly 'input' and 'output'. 'output' has two subfolders, 'pass' and 'fail'. Below you can see several readouts of the directories in question involving the files within. The SORT function scans the input directory for .txt files, ignoring other filetypes. If the text file containts the character '3', the file gets moved to 'output/pass'. If not, they go to 'output/fail'.
